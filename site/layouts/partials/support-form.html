Configuration and log files

You may add debugging files here if you have them. This could include log files or a database dump or any other file used by your system to log error or event messages or record configuration settings.

File types accepted: .cmap, .proj, .log, .dmp, .dat, .arr, .mdmp, .cfg, .txt

You can upload up to 3 files here. To add more than one file, browse to the correct folder and then hold down the [shift] or [control] ([command] on Mac) key and select your additional files.
